headless on openbsd¶

Need to disable gpu

chrome --headless --disable-gpu https://www.google.com

Print to pdf¶

I did this from ~/Downloads to make unveil and pledge happy. Not sure it’s necessary.

chrome --headless --disable-gpu --print-to-pdf=./x.pdf ./index.html
