List guests:¶

virsh list

List all guests:¶

virsh list --all

Power off a guest:¶

virsh destroy HOST

Edit a host’s configuration:¶

virsh edit HOST

List the networks:¶

virsh net-list

Destroy a network:¶

virsh destroy NETWORK

See the dhcp leases:¶

virsh net-dhcp-leases NETWORK

Edit the networks:¶

virsh net-edit NETWORK

Find the vnc display of a guest:¶

virsh vncdisplay gamont
