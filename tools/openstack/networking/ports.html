Ports/Interfaces¶

static ip for instance¶

To set a static IP (or fixed IP) for an instance, create a port with the associated IP address.

$ openstack port create --network NETWORK_NAME --fixed-ip subnet=SUBNET_NAME,ip-address=192.168.1.123 test-static-ip-1

Changing the security group might also be necessary. If so, add the –security-group <security-group> option.
