Only upgrade security related packages¶

Note

From iam-TJ at https://news.ycombinator.com/item?id=11475474

sudo /usr/bin/unattended-upgrade

Note

This will only use the ${dist}-security target. Most systems should have this installed and available. The default configuration is at: /etc/apt/apt.conf.d/50unattended-upgrades It is configured as a system service that runs automatically at shutdown.

Enable a service¶

update-rc.d ssh enable
