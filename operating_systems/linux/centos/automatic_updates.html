Automatic updates:¶

Install the yum-cron package.

# yum install -y yum-cron

Automatic security updates only:¶

Modify /etc/yum/yum-cron.conf and /etc/yum/yum-cron-hourly.conf so the update_cmd is security:

update_cmd = security
