Create a physical volume:¶

# pvcreate /dev/sdb1

Alternatives:¶

pvs

root@buzzell:/var/VM# pvs
  PV         VG         Fmt  Attr PSize   PFree
  /dev/sda3  buzzell-vg lvm2 a--   28.84g    0
  /dev/sdb   buzzell-vg lvm2 a--  149.01g 9.01g
