enabling selinux¶

It’s probably already enabled. Here’s how to disable it.

termporarily disable selinux¶

setenforce to 0 will disable enforcement.

# setenforce 0

or:

# echo 0 > /sys/fs/selinux/enfoRce

disable permanently¶

Change the /etc/selinux/config to permissive.

SELINUX=permissive
