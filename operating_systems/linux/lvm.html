Note

Taken from http://www.howtogeek.com/howto/40702/how-to-manage-and-use-lvm-logical-volume-management-in-ubuntu/ The instructions there give a lot more detail.

View current lvm info¶

# pvdisplay

Create a partition:¶

Apparently not always necessary or something.

# fdisk /dev/sdb

Create a physical volume:¶

# pvcreate /dev/sdb1

Add to the volume group:¶

# lvs
# vgextend buzzell-vg /dev/sdb1

Create a logical volume:¶

# lvcreate -L 50G -n var buzzell-vg

Format and Mount:¶

# mkfs -t ext4 /dev/mapper/buzzell--vg-var
# mount /dev/mapper/buzzell--vg-var /mnt
