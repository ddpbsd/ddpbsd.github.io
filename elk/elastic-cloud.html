elastic-cloud¶

Restart an elasticsearch resource¶

It’s part of the deployments api I guess.

REF_ID is the ref_id in the deployments output. RESOURCE_KIND is elasticsearch, kibana, apm, integrations_server, appsearch, enterprise_search

curl -XPOST -H"Authorization: ApiKey APIKEY" https://api.elastic-cloud.com/api/v1/deployments/DEPLOYMENT_ID/RESOURCE_KIND/REF_ID/_restart
