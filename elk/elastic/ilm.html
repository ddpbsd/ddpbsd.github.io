Index Lifecycle Management¶

I have some mental block with ILM. So trying to get a grasp on it has been difficult.

Basic steps¶

  1. Create index template

  2. Create the Lifecycle policy

  3. Add the lifecycle policy to the template

  4. Bootstrap the index

  5. Rejoice?

Index creation¶

Boostrap the index with the following settings:

{
  "aliases": {
    "INDEX_ALIAS": {
      "is_write_index": true
    }
  },
  "settings": {
    "index": {
      "lifecycle": {
        "origination_date": TODAYS_DATE_IN_UNIX_EPOC
      }
    }

  }
}

I’m not entirely sure about the unix epoc thing, but it seems to work.
