vim
===

tabs as spaces
^^^^^^^^^^^^^^

In the ``.vimrc`` file:

.. code-block:: console

   filetype plugin indent on
   " show existing tab with 4 spaces width
   set tabstop=4
   " when indenting with '>', use 4 spaces width
   set shiftwidth=4
   " On pressing tab, insert 4 spaces
   set expandtab
   
In ``vim``:

.. code-block:: console

   :set tabstop=4
   :set expandtab
