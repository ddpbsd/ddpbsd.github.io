Templates:
^^^^^^^^^^

First update using `pveam`:

.. code-block:: console

   # pveam update

Then list the available templates:

.. code-block:: console

   # pveam available

Next select and download a template:

.. code-block:: console

   # pveam download data2tb ubuntu-16.04-standard_16.04-1_amd64.tar.gz
