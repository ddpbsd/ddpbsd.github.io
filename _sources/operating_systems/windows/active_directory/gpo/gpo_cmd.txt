===================
GPO Client Commands
===================

Just some commands to run on clients for various things.


Force Update of GPO
^^^^^^^^^^^^^^^^^^^

.. code-block:: console

   C:\> gpupdate /force

View the GPO settings applied on a system
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Saves the output to `C:\gpresult.htm`

.. code-block:: console

   C:\>GPResult.exe /h C:\gpresult.htm
