==========
interfaces
==========

set promisc mode
^^^^^^^^^^^^^^^^

.. code-block:: console

   auto eth1
   iface eth1 inet manual
      up ifconfig eth1 promisc up
      down ifconfig eth1 promisc down

turn off power save for wifi
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

   iw wlan0 set power_save off

show the current power save status for wifi
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

   iw wlan0 get power_save
