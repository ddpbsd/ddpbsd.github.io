===
RPM
===


Which RPM contains FILE
^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

   rpm -qf /path/to/FILE

error: "zlib-devel" specifies multiple packages
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

   rpm -q --queryformat "%{name}.%{arch}\n" PACKAGE_NAME

see the files in an rpm
^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

   rpm -ql PACKAGE_NAME
