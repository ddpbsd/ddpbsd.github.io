.. _securityonion_bro_process_control:


Restart bro
^^^^^^^^^^^

.. code-block:: console

   sudo nsm_sensor_ps-restart --only-bro

Stop bro
^^^^^^^^

.. code-block:: console

   sudo nsm_sensor_ps-stop --only-bro

Start bro
^^^^^^^^^

.. code-block:: console

   sudo nsm_sensor_ps-start --only-bro


Which scripts are loaded?
^^^^^^^^^^^^^^^^^^^^^^^^^

Check `/nsm/bro/logs/current/loaded_scripts.log`.
